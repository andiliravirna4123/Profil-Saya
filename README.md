# Website Portofolio A. Lira Virna
Website ini dibuat sebagai mata kuliah dari Pembelajaran Web.

## Deskripsi Proyek
Website portofolio ini berisi informasi tentang diri saya, pendidikan, serta keahlian yang saya miliki.  
Website dibuat menggunakan HTML, CSS, dan JavaScript dengan tampilan sederhana namun modern.

## Fitur Website
- Mode Gelap dan Terang
- Tombol sapa pengguna
- Tampilan responsif
- Animasi sederhana
- Navigasi antar section

## Refleksi Singkat
Mengerjakan proyek ini memberikan saya banyak pelajaran baru dalam pembelajaran web, terutama dalam penggunaan Git dan GitHub.  
Pada awal pengerjaan, saya masih merasa bingung dalam melakukan commit dan push ke repository, namun setelah mencoba beberapa kali, saya mulai memahami alur kerja version control.
Saya juga belajar bahwa pembuatan website tidak hanya tentang tampilan, tetapi juga tentang kerapian struktur kode dan dokumentasi.  
Dengan adanya README ini, saya menyadari pentingnya menjelaskan sebuah proyek agar dapat dipahami orang lain.
Proyek ini membuat saya lebih percaya diri dalam mengelola file proyek dan mengunggahnya ke GitHub, serta menambah motivasi untuk terus belajar pengembangan web.
